FAQ


Как сгенерировать отчет?

1) В меню выберите кнопку в левом меню “Создать новый отчет”

2) В пункте "Название отчета" введите название отчета

3) В пункте Краткое содержание отчета опишите ваши основные пожелания к отчету

4) В пункте Тип отчета выберите один из предложенных шаблонов (вы сможете редактировать их в пункте меню “Шаблоны отчетов”, подробнее в главе инструкции “Как настраивать шаблоны?”)


5) В поле Дополнительные источники вы можете добавить источники в формате через запятую (например, domain1.ru, domain2.ru), которых не было в шаблоне! Внимание, каждый шаблон уже содержит предустановленные источники, проверить которые можно во вкладке “Шаблоны”

6) В поле Фильтры вы можете написать условия поиска, которые будут использованы как при поисковых запросах, так и при генерации отчета (напр. “Не вноси в отчет данные о проектах из компании N”)

7) В поле Начальная дата укажите с какого периода вы хотите рассматривать информацию для отчета. Данные вводить в формате. ГГГГ-ММ-ДД

8) В поле Конечная дата укажите до какого периода вы хотите рассматривать информацию для отчета. Данные вводить в формате. ГГГГ-ММ-ДД

9) В поле Данные для графиков добавьте файлы, необходимые для построения графиков

10) В поле Дополнительные файлы в формате PDF добавьте необходимые файлы для анализа

11) Нажмите Загрузить данные для проверки и отправки данных на сервис

12) В следующем окне вы увидете введенные вами данные для генерации отчета. Для генерации отчета нажмите Сгенерировать отчет.

*Время генерации отчета может составлять до 20 минут.


Как вносить изменения в сгенерированный отчет?

1) Откройте готовый отчет

2) Найдите главу, которую хотите исправить и нажмите в ее конце кнопку“Редактировать главу” (см. скриншот)


3) В открывшемся окне внизу страницы выберите нужный вам вариант

- “Перегенировать главу” перезапустит LLM для данной главы

- “Изменить главу” позволяет вносить изменения в отчете в ручном режиме


Как настраивать шаблоны?

1) Перейдите в пункт меню “Шаблоны” отчетов

2) Выберите отчет в который вы хотите внести правки (или посмотретьинформацию о деталях отчета)

3) Выберите пункт “Изменить данные о шаблоне отчета” внизу страницы

) Внесите необходимые изменения.- В пункте “Sources” вы можете выбрать все необходимые вамисточники зажимая Ctrl (Как добавить источники в этот списокможно узнать в главе инструкции “Как вносить источники для шаблонов?”)

- В пунтке “Chapters” можно менять структуру глав и подглависпользуя следующую структуру текста: [ “Название главы 1:Подглава_1, Подглава_2”, “Название главы 2: Подглава_1,Подглава_2, Подглава_3”]

- В пунктах “Chapter prompt”, “Prompt search”, “Prompt search” используются оптимальные для нашего решения промты, одинаковы также можете вносить в них коорективы! ВНИМАНИЕ при создании кастомного отчета рекомендуется вставлять в поля “Chapter prompt”, “Prompt search”, “Prompt search” предзаписанные промты сайта из классических отчетов


Как вносить источники для шаблонов?

1) В меню выберите пункт “Источники”

2) В появившемся спсике выберите “Добавить источник”

3) Зполните поля: “Source” для названия источника и “URL” для ссылки(поле “Source feature” опционально, можно вставить “-” при отсутствииинформации для ввода)

4) Нажмите “Submit” для сохранения источника

*)Вы можете вносить в существующие источники, для этого просто кликните нанего в списке источников
